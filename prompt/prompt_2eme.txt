OBJECTIF DE CETTE ÉTAPE

Tu es dans une phase STRICTEMENT ANALYTIQUE.

Tu ne dois PAS :
- prendre de décision finale
- choisir une architecture
- recommander une solution
- fusionner les opinions
- produire une synthèse décisionnelle

Tu dois uniquement :
- analyser les relations entre sources
- comparer les informations
- détecter les structures communes et divergentes
- extraire des signaux techniques exploitables
- identifier les biais d’analyse

--------------------------------------------------

SORTIE ATTENDUE

## 1. Consensus global
Liste des points confirmés par toutes les sources sans exception.

## 2. Divergences
Différences entre les sources + hypothèses expliquant ces divergences.

## 3. Signaux techniques fiables
Informations techniques récurrentes et robustes (indépendantes de l’opinion).

## 4. Zones d’incertitude
Points non couverts, ambiguïtés, ou informations manquantes critiques.

## 5. Biais détectés
Analyse des biais possibles :
- sur-optimisation
- sur-simplification
- sur-ingénierie
- confusion entre concepts (ex : identity vs dedup)
- biais de langage ou d’architecture
