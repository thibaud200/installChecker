PIPELINE INVARIANTS



1\. SOURCE REALITY RULE

\- Only raw sources are truth input



2\. TRACE AUDITOR RULE

\- Produces EVIDENCE\_MAP only

\- No scoring, no decision



3\. SCORING RULE

\- Uses only EVIDENCE\_MAP + metadata

\- No reinterpretation allowed



4\. DECISION RULE

\- Uses only FINAL\_SCORING output

\- No access to raw sources



5\. ANTI-DOUBLE-COUNT RULE

\- A signal may only exist once in the pipeline

\- Any derived form is non-authoritative



6\. NO CROSS-LAYER REASONING RULE

\- No layer can simulate reasoning of another layer
